# Travelgram - A Mock-Up Social Network Site (v3.1.0)

This repository creates a fully functional  of a mock-up social network site called "Travelgram", which looks very much like Instagram. Users can create a profile, scroll a feed and explore page, click on posts, interact with them (likes and comments) and also browse user profiles. They can share their own images and save other people's posts to their profiles.  

It is written in react and javascript, utilizing v0.
